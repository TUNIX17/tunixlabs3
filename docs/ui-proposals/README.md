# TunixLabs - UI Proposals 2026

6 propuestas de diseno para la pagina de inicio basadas en las tendencias de UI/UX 2026.

## Como visualizar

Abre cualquiera de los archivos HTML en tu navegador para ver la propuesta.

---

## Propuestas

### 1. Neumorfismo (Soft UI) - `01-neumorphism.html`

**Tendencia:** Neumorphism 2.0 / Soft UI

Diseno con sombras suaves que crean una sensacion tactil y 3D. Esta version 2026 mejora la accesibilidad con mejor contraste respecto al neumorfismo original de 2020.

**Caracteristicas:**
- Sombras suaves externas e internas
- Elementos que parecen "elevados" o "hundidos"
- Paleta clara y neutral
- Botones e inputs con efecto 3D sutil
- WCAG compliant con mejor contraste

---

### 2. Bento Grid - `02-bento-grid.html`

**Tendencia:** Bento Grid Layout

Layout modular inspirado en las cajas bento japonesas. Perfecto para mostrar multiples tipos de contenido de forma organizada y visualmente dinamica.

**Caracteristicas:**
- Grid asimetrico con tarjetas de diferentes tamanos
- Modo oscuro con acentos de color
- Microinteracciones en hover
- Jerarquia visual clara
- Ideal para mostrar servicios y estadisticas

---

### 3. Glassmorphism + Aurora - `03-glassmorphism-aurora.html`

**Tendencia:** Glassmorphism + Aurora Gradients

Combina el efecto de vidrio esmerilado con fondos de gradientes organicos animados tipo aurora boreal. Muy popular en apps premium.

**Caracteristicas:**
- Efecto blur y transparencia en tarjetas
- Fondo con blobs animados (aurora)
- Gradientes vibrantes
- Sensacion premium y moderna
- Navegacion flotante con glass effect

---

### 4. Dark Mode Minimal - `04-dark-minimal.html`

**Tendencia:** Minimal Dark + Grid System

Diseno ultra minimalista con fondo negro puro. Enfocado en tipografia, espacios y una estetica profesional sin distracciones.

**Caracteristicas:**
- Fondo negro (#000000)
- Tipografia como elemento principal
- Bordes sutiles
- Sin gradientes ni efectos excesivos
- Grid background sutil
- Numeracion de servicios estilo corporativo

---

### 5. 3D Immersive - `05-3d-immersive.html`

**Tendencia:** 3D Spatial Design + Immersive

Inspirado en experiencias de realidad extendida (XR). Elementos 3D flotantes, particulas y sensacion de profundidad espacial.

**Caracteristicas:**
- Fondo oscuro con estrellas
- Formas 3D flotantes (cubos, esferas, anillos)
- Animaciones de rotacion y flotacion
- Efecto de luz y brillo (glow)
- Botones con efecto de luz deslizante
- Inspiracion sci-fi/espacial

---

### 6. Liquid Glass + Kinetic Typography - `06-liquid-glass-kinetic.html`

**Tendencia:** Liquid Glass + Kinetic Typography

La tendencia mas moderna de 2026, inspirada en el nuevo lenguaje de diseno de Apple. Combina superficies liquidas con tipografia animada.

**Caracteristicas:**
- Fondo con blobs liquidos que cambian de forma
- Glass effect con alta saturacion
- Tipografia kinetica (palabras que flotan)
- Variable fonts con animacion de peso
- Gradientes de texto animados
- Botones con efecto ripple liquido
- Orbitas animadas alrededor del robot
